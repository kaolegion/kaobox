# KaoBox

KaoBox est une infrastructure agentique modulaire.

## Structure

- bin/        → CLI et interfaces
- core/       → moteur système
- base/       → manifests système
- modules/    → extensions modulaires
- state/      → état runtime
- logs/       → journaux
- doc/        → documentation officielle

## Philosophie

KaoBox est conçu comme un brain minimaliste,
modulaire, déterministe et extensible.

## Version

Brain v1 - Initial structured architecture
